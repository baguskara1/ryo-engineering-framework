# Checklist

- [ ] Problem constraints analyzed and algorithm chosen accordingly
- [ ] Time complexity estimated and within limits
- [ ] Edge cases considered (empty, 1, max, min, duplicates)
- [ ] Input/output format matches specification
- [ ] Faster I/O used (sync_with_stdio, sys.stdin.buffer)
- [ ] Data structures chosen for efficiency
- [ ] Memory usage estimated and within limits
- [ ] Test with sample cases before submission
- [ ] Modular arithmetic used when results could be large
- [ ] Debugging with small custom test cases
