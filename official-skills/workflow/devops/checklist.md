# Checklist

- [ ] Infrastructure defined as code (Terraform, Pulumi, CloudFormation)
- [ ] Configuration management automated (Ansible, Chef, Puppet)
- [ ] Monitoring and alerting configured for all services
- [ ] Centralized logging with ELK/Loki stack
- [ ] Disaster recovery plan documented and tested
- [ ] Secrets managed with vault solution (HashiCorp Vault, AWS Secrets Manager)
- [ ] CI/CD pipelines include infrastructure validation
- [ ] Runbooks documented for common operational tasks
- [ ] Cost monitoring and budget alerts configured
- [ ] Security patches applied regularly with automated scanning
