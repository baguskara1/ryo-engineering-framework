# Checklist

- [ ] Data split into train/validation/test sets
- [ ] Features scaled/normalized appropriately
- [ ] Missing values handled (imputation or removal)
- [ ] Categorical variables encoded correctly
- [ ] Cross-validation used for model evaluation
- [ ] Hyperparameter tuning performed (GridSearchCV, Optuna)
- [ ] Model performance metrics match business objectives
- [ ] Experiment tracking with MLflow or similar
- [ ] Model versioned and stored in model registry
- [ ] Deployment pipeline includes monitoring for drift
