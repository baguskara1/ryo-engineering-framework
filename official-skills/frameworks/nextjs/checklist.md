# Checklist

- [ ] App Router used for new projects (not Pages Router)
- [ ] Server components preferred for data fetching
- [ ] Client components marked with "use client" directive
- [ ] Metadata API used for SEO optimization
- [ ] Image component with proper sizing and lazy loading
- [ ] Route handlers for API endpoints with error handling
- [ ] Middleware configured for auth/redirects
- [ ] Static and dynamic rendering used appropriately
- [ ] Environment variables prefixed with `NEXT_PUBLIC_` for client access
- [ ] Bundle analyzer configured to monitor build size
