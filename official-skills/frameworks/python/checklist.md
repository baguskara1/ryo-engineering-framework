# Checklist

- [ ] Virtual environment activated and dependencies in `requirements.txt` or `pyproject.toml`
- [ ] Type hints used for function signatures
- [ ] Code formatted with Black and linted with Ruff
- [ ] Tests written with pytest and test coverage checked
- [ ] Async/await used for I/O-bound operations
- [ ] Environment variables managed via `python-dotenv` or `pydantic-settings`
- [ ] Logging configured with appropriate levels
- [ ] `.gitignore` includes `__pycache__/`, `.venv/`, `*.pyc`
- [ ] Python version specified in `.python-version` or `pyproject.toml`
- [ ] Optional static type checking with mypy or pyright
