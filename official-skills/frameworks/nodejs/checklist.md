# Checklist

- [ ] Use modern ES modules or CommonJS consistently
- [ ] Async/await used for all asynchronous operations
- [ ] Error handling with try/catch and global error handlers
- [ ] Environment variables managed via dotenv or similar
- [ ] Package.json scripts defined for dev, build, start, test
- [ ] Dependencies audited for vulnerabilities (`npm audit`)
- [ ] Streams used for large file/data processing
- [ ] Process signals handled for graceful shutdown
- [ ] Debugging with `--inspect` flag or VS Code debugger
- [ ] Node version managed via .nvmrc or engines field
