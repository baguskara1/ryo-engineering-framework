# Checklist

- [ ] Models use `$fillable` or `$guarded` for mass assignment protection
- [ ] Routes follow RESTful conventions with resource controllers
- [ ] Validation logic extracted to Form Request classes
- [ ] Database migrations version controlled for schema changes
- [ ] Queues configured for long-running or background tasks
- [ ] Caching configured for expensive queries and views
- [ ] API authentication implemented with Sanctum or Passport
- [ ] Horizon used for queue monitoring in production
- [ ] Telescope configured for local debugging
- [ ] Horizon/Telescope secured in production environments
