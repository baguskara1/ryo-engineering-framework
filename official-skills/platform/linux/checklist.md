# Checklist

- [ ] Filesystem permissions follow least privilege principle
- [ ] Unnecessary services disabled and removed
- [ ] Firewall configured (ufw/iptables/nftables)
- [ ] SSH configured with key-based auth and disabled root login
- [ ] Automatic security updates enabled
- [ ] Audit logging configured (auditd)
- [ ] SELinux or AppArmor enabled and enforcing
- [ ] Swap configured appropriately for workload
- [ ] Log rotation configured with logrotate
- [ ] System backups scheduled and tested
