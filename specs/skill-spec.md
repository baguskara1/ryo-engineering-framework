# Skill Specification v1.0

Every skill inside the framework MUST follow the same structure.

## Required Files

```
manifest.yaml
metadata.yaml
README.md
SKILL.md
examples.md
checklist.md
decision-tree.md
troubleshooting.md
references.md
CHANGELOG.md
VERSION.md
```

## Required Folder

```
prompts/
```

Inside prompts:

```
system.md
review.md
generate.md
```

## Goals

- Consistent
- Easy to maintain
- Easy for AI to understand
- Easy to validate automatically
