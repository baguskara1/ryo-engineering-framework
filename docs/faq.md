# FAQ

## General

### What is RYO Engineering Framework?

RYO is an open-source engineering framework that provides reusable skills, playbooks, templates, and standards for AI coding assistants.

### Who is RYO for?

Developers and teams who want consistent, production-ready software engineering workflows powered by AI.

### How is RYO different from other AI coding tools?

RYO provides structured engineering skills with metadata, manifests, checklists, and prompts — not just raw prompts.

## CLI

### How do I install a skill?

Run `ryo install <skill-name>`. The skill will be downloaded from the official registry.

### How do I create my own skill?

Run `ryo create <category> <skill-name>` to scaffold a new skill from a template.

### Can I publish my skills?

Yes, use `ryo publish <skill-name>` to package a skill for distribution.

## Skills

### What skills are available?

Run `ryo registry` or `ryo search <keyword>` to discover available skills.

### How do I update a skill?

Run `ryo update <skill-name>` to update an installed skill.
