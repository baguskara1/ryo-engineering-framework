# Common Problems

## Styles Not Applied

Possible Causes

- Wrong content paths
- Build cache
- Missing Tailwind directives

Solution

- Verify content configuration
- Restart dev server

---

## Large class lists

Possible Causes

- Repeated utility combinations

Solution

- Extract reusable component

---

## Responsive Layout Broken

Possible Causes

- Incorrect breakpoint usage

Solution

- Follow mobile-first approach
