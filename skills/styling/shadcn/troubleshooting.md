# Common Problems

## Styles Missing

Possible Causes

- Tailwind content paths incomplete
- Components not installed correctly

Solution

- Verify Tailwind configuration
- Re-run shadcn add

---

## Variant Not Working

Possible Causes

- Incorrect cva configuration

Solution

- Check variant definitions

---

## Hydration Warning

Possible Causes

- Browser-only APIs inside Server Components

Solution

- Move interactive logic to Client Components
