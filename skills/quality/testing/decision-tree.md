# Testing Decision Tree

Need to test pure logic?

↓

Unit Test

---

Need to test multiple components/services together?

↓

Integration Test

---

Need to test the full user flow?

↓

End-to-End Test

---

Need to mock dependencies?

↓

Mock external systems only

Avoid mocking business logic

---

Found a bug?

↓

Write a failing test

↓

Fix the implementation

↓

Verify the test passes
