# Testing Checklist

## Coverage

- [ ] Critical business logic tested
- [ ] Edge cases covered
- [ ] Error scenarios tested

## Quality

- [ ] Tests are deterministic
- [ ] Tests are isolated
- [ ] No flaky tests

## Readability

- [ ] Clear test names
- [ ] Arrange-Act-Assert pattern
- [ ] Minimal mocking

## CI

- [ ] Tests pass locally
- [ ] Tests pass in CI
