# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in RYO Engineering Framework, please report it privately by opening a security advisory on GitHub.

We will acknowledge receipt within 48 hours and provide a timeline for resolution.

## Scope

Security issues in the CLI tool, registry, or official skills are in scope. Issues in third-party dependencies should be reported to their respective maintainers.

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Best Practices

- Always use the latest version
- Review skills before installing from untrusted sources
- Report suspicious packages
